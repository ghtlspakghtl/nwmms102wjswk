🚨 전원이 안 켜짐

  1. 뒤판의 전원 플러그 연결 상태를 확인해주십시오.
  2. 멀티탭을 사용하고 있다면, 멀티탭이 작동하는지 확인해주십시오.
  3. 전원 버튼을 눌러도 불빛이 안 들어온다면 플러그나 멀티탭의 작동 유무를 확인해주세요.

🔌 HDMI 연결 불가

  1. 전면 HDMI 와 전면 C타입의 연결단자가 훼손되 있습니다. 전면이 아닌 뒷면으로 해주십시오.
  2. 뒷면 HDMI는 1과 2로 나뉘어 있을 거고 HDMI(숫자) 수업용 이라 적힌 것을 눌러 주세요. 방송실은 방송실을 위한것입니다. (단 4번 항목의 조건일 경우 수정이 필요)
  3. 노트북의 주사율(Hz) 와 해상도를 변경해주세요.
  4. 뒷면의 연결 단자는 가끔 뽑힐 경우가 있습니다. (만약 단자의 문제일 경우 방송실과 수업용의 순서를 변경하십시오.)

📝 화이트 보드 사용불가

이런 경우는 시스템상의 문제입니다. 전원을 껐다 켜주시고 안 될 경우 담당자에게 문의 바랍니다.

🌐 인터넷 사용 불가

  1. USB 타입으로 된 IPTIME 연결기를 쓰고 있는지 확인해주세요.
  2. USB 타입 IPTIME 연결기에 파란 불빛이 들어오는지 확인해주세요.
  3. UPS라고 적혀져 있는 버튼을 눌러주세요.

🖼️ 브라우저 사용 불가

  1. 전자칠판의 윈도우 버전의 크롬을 사용해주십시오.
  2. 학교 망에서 차단한 경우가 있습니다. (이런 경우 난우중 전산실이나 담당 선생님한테 문의 바랍니다.)
